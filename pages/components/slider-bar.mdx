# SliderBar

Animated slider with gesture control, step snapping, labels, and haptic feedback.

- **Tier 3** — requires `react-native-reanimated` + `react-native-gesture-handler`
- **Registry deps:** [Theme](/theme), [Haptics](/utils/haptics), [IconSymbol](/components/icon-symbol)
- **Platforms:** iOS, Android

```bash [Terminal]
npx popapp add slider-bar
```

## Usage

```tsx
import { SliderBar } from "@/components/ui/slider-bar";

<SliderBar
  value={volume}
  onValueChange={setVolume}
  min={0}
  max={1}
/>
```

## With step snapping

```tsx
<SliderBar
  value={rating}
  onValueChange={setRating}
  min={1}
  max={5}
  step={1}
/>
```

## With labels

```tsx
<SliderBar
  value={intensity}
  onValueChange={setIntensity}
  labels={[
    { text: "Low", position: 0, icon: "tortoise.fill" },
    { text: "Medium", position: 0.5 },
    { text: "High", position: 1, icon: "hare.fill" },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | — | Current value |
| `onValueChange` | `(value: number) => void` | — | Change handler |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `1` | Maximum value |
| `step` | `number` | `0` | Step size (0 = continuous) |
| `labels` | `SliderLabel[]` | — | Position labels with optional icons |
| `haptic` | `boolean` | `true` | Haptic feedback on step changes |
