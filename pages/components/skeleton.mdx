# Skeleton

Pulsing loading placeholder with configurable dimensions.

- **Tier 2** — requires `react-native-reanimated`
- **Registry deps:** [Theme](/theme)

```bash [Terminal]
npx popapp add skeleton
```

## Usage

<PreviewImage src="/images/skeleton.png" alt="Skeleton loading" />

```tsx
import { Skeleton } from "@/components/ui/skeleton";

{/* Text placeholder */}
<Skeleton width="60%" height={16} />

{/* Avatar placeholder */}
<Skeleton width={48} height={48} borderRadius={24} />

{/* Card placeholder */}
<Skeleton width="100%" height={120} borderRadius={12} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `width` | `number \| %` | — | Skeleton width |
| `height` | `number` | — | Skeleton height |
| `borderRadius` | `number` | `8` | Corner radius |
