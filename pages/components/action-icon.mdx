# ActionIcon

Circular icon button with Liquid Glass support and size/variant options.

- **Tier 1** — no animation dependencies
- **Registry deps:** [Theme](/theme), [Glass](/utils/glass), [IconSymbol](/components/icon-symbol)

```bash [Terminal]
npx popapp add action-icon
```

## Usage

<PreviewImage src="/images/action-icons-variations.png" alt="ActionIcon variations" />

```tsx
import { ActionIcon } from "@/components/ui/action-icon";

<ActionIcon name="plus" onPress={handleAdd} />
<ActionIcon name="heart.fill" onPress={handleFavorite} />
<ActionIcon name="trash" variant="solid" size="lg" onPress={handleDelete} />
<ActionIcon name="square.and.arrow.up" onPress={handleShare} />
```

## Sizes

```tsx
<ActionIcon name="plus" size="xs" />  {/* 32px */}
<ActionIcon name="plus" size="sm" />  {/* 38px */}
<ActionIcon name="plus" size="md" />  {/* 44px */}
<ActionIcon name="plus" size="lg" />  {/* 56px */}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `IconSymbolName` | — | SF Symbol name |
| `variant` | `"solid" \| "outline" \| "ghost" \| "subtle" \| "destructive"` | `"ghost"` | Visual style |
| `size` | `"xs" \| "sm" \| "md" \| "lg"` | `"md"` | Button size |
| `disabled` | `boolean` | `false` | Disable interaction |
| `isLoading` | `boolean` | `false` | Show loading spinner |
| `haptic` | `boolean` | `true` | Haptic feedback |
| `glass` | `boolean` | `true` | Liquid Glass (iOS 26+) |

Extends `TouchableOpacityProps`.
